# 市电电力故障	

## 提前准备	

1. 资源准备
（1）移动自有站点：派单对象移动代维团队，响应时限（紧急30 分钟 / 重要 1 小时 / 一般2 小时）；
（2）铁塔站点：派单对象铁塔代维团队，响应时限（紧急30 分钟 / 重要 1 小时 / 一般2 小时）；
（3）资源准备：警示锥、警示带、警示牌（“电力作业，禁止合闸”）、万用表、发电机（15kW）、4*6电源线。
2. 安全准备
（1）操作人员需持有效且在有效期内的电工证（涉及电源操作）和登高证（作业高度大于或等于2米）；
（2）配备安全防护用品：绝缘梯、绝缘手套及绝缘鞋（耐压≥1000V，在有效期内）、安全帽（在有效期内）、双钩安全带（承重≥15kN，配备缓冲器）、数字验电笔（支持12V-220V）、反光衣、急救包（含触电急救用品、止血带、碘伏等基础用品）；
（3）作业前开展安全技术交底，明确风险点（如高空坠落、触电、设备坠落）及应急措施（如触电急救流程、高空救援方案）。
3. 权责划分与协同机制
（1）移动自有站点：移动代维全责，需同步通知移动基站接口人（如站点物业）；
（2）铁塔站点：铁塔代维负责市电接入至 ATS，移动代维负责负载侧检查，双方通过 “站址协同群” 共享进度；
（3）跨部门协调：若涉及供电部门停电，移动自有站点由移动对接供电所，铁塔站点由铁塔区域经理协调，结果同步至对方。"
        
        
##  现场操作	

1. 共性进站与安全规范
无论何种站点，抵达后均需：
（1）与接口人签署《基站出入登记单》（注明站点类型、作业内容）；
（2）划定作业区域（设置围栏、警示牌），执行 “双人操作制”（一人操作，一人监护）；
（3）室外作业避开雷雨 / 暴雨天气，高压区域（如变压器）需保持安全距离（≥1.5 米）。
2. 移动自有站点排查与处理
步骤 1：市电链路检测（自外而内）
室外总配电箱：测量输入端电压（L-N 正常 220V±10%），检查空开是否跳闸（若跳闸，测输出端绝缘电阻≥0.5MΩ 后试合 1 次）；
机房市电屏：测量输入 / 输出电压，检查熔丝（如 32A）是否熔断、接线端子是否松动（紧固扭矩：M6 端子 8-10N・m）；
整流机架状态：确认市电异常时是否自动切换至蓄电池（监测蓄电池电压：48V 系统正常 53.5-54.5V）。
步骤 2：故障处理与应急供电
若为线路松动 / 氧化：断电后打磨端子、涂导电膏，紧固后用红外测温仪测温度（≤60℃）；
若为市电中断且蓄电池续航＜1 小时：使用发电机进行发电，连接至应急供电端子（核对相序），确保输出电压 380V±5%、频率 50Hz±0.5Hz；
若为过压 / 欠压：启用移动稳压器（30kVA），待电网恢复正常后切换回市电。
3. 铁塔站点排查与处理（铁塔 + 移动协同）
铁塔代维操作（市电侧）：
按铁塔规范检测：高压变压器（低压侧 380V±10%）→ 铁塔室外总配电箱（电压、空开状态）→ 机房铁塔市电屏（防雷模块、端子电压）；
故障修复：空开跳闸按铁塔流程试合，接头松动按力矩标准紧固（M8 端子 25N・m），记录铁塔动环系统数据。
移动代维操作（负载侧）：
确认整流机架或一体化电源输出端电压（正常后检查负载设备供电）；
若铁塔市电恢复但负载无电，排查整流机架或一体化电源至 BBU/RRU 的电缆、空开（属移动维护范围）。
应急发电协同：
铁塔代维申请铁塔发电机，发电机电压正常或整流机架输出正常，双方共同确认供电正常（记录发电开始时间、油位）。
4. 共性验证标准
市电恢复后，需满足：
电压：单相 220V±10%，三相 380V±10%；
频率：50Hz±0.5Hz；
告警：移动 / 铁塔动环系统 “市电故障” 告警清除；
负载：BBU、RRU 等设备运行正常（移动侧验证），蓄电池开始浮充（电流≤5A）。"

##  闭环管理	
1. 现场收尾
移动自有站点：
恢复设备原状（锁好配电箱、市电屏），清理工具，通知接口人市电已恢复；
若使用固定发电机，停机前需先切换回市电，怠速 3 分钟后关闭，补充燃油至少80%；
若使用移动油机，停机前需先切换回市电，怠速 3 分钟后关闭，需清空燃油至油桶。
铁塔站点：
铁塔代维检查市电链路及负载供电，通知接口人市电已恢复；
铁塔代维回收工具，移动代维确认负载设备正常，铁塔代维关闭现场警示标识。
2. 工单闭环与台账更新
（1）移动自有站点，登录掌上运维 APP，进入代维掌中宝 - 故障工单：站点市电故障已修复，设备运行正常，告警清除。如有发电做好发电工单回复。
（2）铁塔站点，铁塔运维系统《铁塔市电处理单》、协同确认单、动环告警清除截图
3. 复盘与改进
对 30 天内重复故障（≥2 次）：
移动自有站点：移动组织分析会，输出《整改报告》（如更换老化电缆、加装防雷器）；
铁塔站点：铁塔牵头，移动参与，联合制定优化方案（如升级铁塔配电箱空开）。"
