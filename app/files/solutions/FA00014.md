# 时钟盒/GPS故障	

## 提前准备	
1. 告警接收与关联分析​
通过“无线网管获取 “时钟同步异常”“GPS 失锁”“时钟源不可用” 等告警”（含告警代码，如 “GPS 失锁告警 0x000A”“系统时钟源切换告警 0x0012”），同步核查关联现象：​
（1）时钟告警特征：小区间出现切换失败（切换成功率＜80%）、同频干扰（RSRP 正常但 SINR＜-3dB）、UE 接入失败（RRC 连接成功率骤降）；​
（2）GPS 故障特征：GPS 天线无锁定（BBU 面板 GPS 灯灭或闪烁）、定位信息丢失（网管显示 “经纬度未知”）、依赖 GPS 的时钟源（如 1588v2）同步失败。​
（3）排除非时钟 / GPS 因素：确认传输链路正常（无断链、误码）、基带板运行正常（无硬件故障告警），且故障仅与时间同步相关（如不同步小区的时钟偏差＞1.5μs）。​
2. 时钟源与 GPS 配置确认​
通过网管系统查询 BBU 时钟配置：​
（1）主时钟源：“参考时钟源类型=GNSS Clock，参考时钟源状态=可用，参考时钟源激活状态=激活""；​
（2）同步层级：核心网→汇聚层 PTN→接入层 PTN→BBU（时钟同步链路拓扑）；​
（3）GPS 参数：""天线型号（如华为 GPS-ANT-001）、安装位置（铁塔顶 / 机房顶，视野无遮挡）、馈线长度（≤50m，衰减≤10dB）""。​
3. 故障边界划分​
依据《无线时钟同步维护规范》，明确责任范围：​
（1）无线维护责任：BBU 时钟板、GPS 天线及馈线（含避雷器）、本地时钟配置；​
（2）传输维护责任：传输侧 1588v2 时钟源配置、传输链路时钟偏移（需≤100ns）；​
若 GPS 天线至 BBU 的馈线通路故障、时钟板硬件异常，或本地时钟源切换逻辑错误，判定为无线侧时钟 / GPS 故障；若传输时钟源异常，需协同传输维护处理。
4. 安全准备
（1）资质要求：操作人员需持有效且在有效期内的电工证（涉及电源操作）和登高证（铁塔 / 抱杆安装场景）；
（2）通用防护：安全帽（冲击吸收性能≤4900N）、反光衣（荧光亮度≥300cd/m²）、急救包（含止血贴、碘伏、纱布、止血带）；
（3）高空防护：双钩安全带（承重≥15kN，配备缓冲器，有效期内）、绝缘梯（梯脚防滑垫硬度≥60 Shore A）；
（4）电气防护：绝缘手套 / 绝缘鞋（耐压≥1000V，试验有效期内）、数字验电笔（适配直流 - 12V 及交流 220V）；
（5）安全交底：作业前明确风险点（如射频接口划伤、高空器件坠落、驻波仪摔损）及应急措施（如划伤后用碘伏消毒、高空坠落时安全带急救流程）。"

## 现场操作	
1. GPS 故障排查与处理​
步骤 1：GPS 天线与馈线检查​
现场查看 GPS 天线安装：是否被遮挡（如树木、建筑物）、是否松动（用扳手检查固定支架，扭矩 8-10N・m）、防水胶是否老化（馈线接头需缠绕 3 层防水胶带，重叠率≥50%）；​
测试馈线衰减：用驻波仪测量 GPS 馈线驻波比（正常≤1.5），用功率计测试天线端接收功率（正常 - 130~-110dBm），若衰减＞10dB，检查馈线是否破损、接头是否氧化（用酒精棉清洁后重新紧固）。​
步骤 2：BBU 侧 GPS 模块测试​
检查 BBU 时钟板 GPS 接口：用万用表测量供电电压（正常 5V±0.5V），若电压异常，排查供电空开（是否跳闸）或接口松动；​
更换 GPS 模块：若接口供电正常但仍失锁，佩戴防静电手环，更换同型号 GPS 模块（如华为 U2020-GPS-MOD），重启后观察锁定状态（GPS 灯常亮表示锁定，需接收≥4 颗卫星）。​
步骤 3：极端场景应急处理​
若 GPS 完全失效（如暴雨导致天线损坏），通过 LMT 将时钟源手动切换至 “传输同步（1588v2）” 或 “本地晶振”，限制小区最大覆盖半径（≤500m）以减少不同步干扰，同时申请紧急更换天线。​
2. 时钟告警排查与处理​
步骤 1：时钟源状态核查​
登录 LMT 查询时钟源优先级：确认主时钟源（如 GPS）是否正常，备用时钟源（如传输同步）是否就绪；​
测试时钟偏移：通过网管查看 BBU 与核心网的时钟偏差（正常≤1.5μs），若偏差＞5μs，触发 “时钟同步告警”，需重新校准。​
步骤 2：传输时钟同步验证（协同传输）​
若主时钟源为传输 1588v2，通知传输维护测试传输链路时钟质量：​
传输侧：检查 PTN 设备时钟板状态（无 “时钟失锁” 告警）、1588v2 报文时延（≤100ns）；​
无线侧：在 BBU 侧抓取 1588v2 报文，确认报文无丢失（丢包率 = 0）、时间戳准确。​
步骤 3：时钟板故障处理​
若时钟源正常但 BBU 仍告警，检查时钟板指示灯（RUN 灯闪烁频率异常表示故障），测量时钟板供电（-48V±5%）；​
更换时钟板：用专用工具拔出故障时钟板，插入同型号备用板（需与 BBU 硬件版本兼容），重启后通过 LMT 重新配置时钟参数（优先级、同步阈值）。​
3. 恢复验证标准​
GPS 恢复：卫星锁定数量≥4 颗，定位精度≤10m，持续锁定 30 分钟无告警；​
时钟同步：BBU 与核心网时钟偏差≤1.5μs，小区切换成功率≥98%，SINR≥-3dB，无同频干扰告警。"
    
## 闭环管理	
1. 跨专业协同（传输 ）​
与传输协同：若传输时钟源异常，传输维护修复后，需网管重新同步时钟（路径：设备管理→时钟管理→手动同步），并反馈同步结果；​
2. 工单闭环与文档记录​
登录掌上运维 APP，进入代维掌中宝 - 故障工单，回复内容：如 “GPS 天线馈线接头氧化”“时钟板硬件故障”；​处理措施（更换备件型号、同步配置参数）；​
3. 预防措施​
    * 定期巡检：巡检或验收时检查 GPS 天线（无遮挡、防水完好）、馈线接头（无氧化），测试时钟偏移量；​
    * 冗余配置：核心区域 BBU 配置 “GPS + 北斗” 双模定位，避免单星座失效；​
    * 告警优化：在网管设置 “GPS 卫星数量＜4 颗” 预告警，提前介入处理。"
